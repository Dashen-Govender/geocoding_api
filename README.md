1. Register a google maps api

   Get the following api's:
   Places API (New),
   Geocoding API,
   Geolocation API,
   Directions API,
   Maps JavaScript API,
   Places API,
   Distance Matrix API   
2. Install the following packages:

   globally: nodemon

   locally: axios,
   body-parser,
   ejs,
   express
3. run npm i
4. run nodemon index.js
